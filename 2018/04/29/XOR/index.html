XOR ^

XOR 异或

一般表示

$a⊕b$

In Python/ C++

a^b

运算性质

  1. 交换律 ：$A ⊕ B = B ⊕ A$
  2. 结合律：$A ⊕ (B ⊕ C) = (A ⊕ B) ⊕ C$
  3. 恒等律：$A ⊕ 0 = A$
  4. 归零率：$A ⊕ A = 0$
  5. 推论：$(B ⊕ A) ⊕ B = A ⊕ (B ⊕ B) = A ⊕ 0 = A$
